Selected GLTF models:

- Blue/red castle: HQ
- Blue/red barracks: melee spawner
- Blue/red archery range: ranged spawner
- Blue/red catapult tower: defense tower
- Blue/red tower B: dragon lair
- Rock A/B/C and crate A big: deterministic obstacle variants

Each GLTF is stored with its referenced BIN and texture. Android export excludes this source folder; only baked PNG atlases ship.
